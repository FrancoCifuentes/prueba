¿Qué significa HTML y por qué se dice que es un lenguaje de marcas? Enumera 8 etiquetas HTML?

HTML es el diminutivo de "HyperText Markup Language", un lenguaje que se utiliza para el desarrollo de páginas de Internet. Se dice que es un lenguaje de marcas ya que no permite la implementación de algoritmos y no dispone de funciones aritméticas ni de variables. Unicamente se combina el texto con etiquetas que contienen información sobre la estructura del texto.
